# 무료 다국어 학습 숏츠 생성기 - 깔끔한 교육용 고정 인트로 버전

OpenAI API 없이 작동하는 무료 버전입니다.  
이번 버전은 **그림체를 하나로 고정**한 버전이며, 스타일은 **깔끔한 교육용**으로 설정되어 있습니다.

브랜드명과 로고 텍스트는 제거했습니다.  
영상에는 1초 인트로 카드와 고정 제목 박스만 남겨 더 깔끔하게 보이도록 했습니다.

## 새로 적용된 고정 요소

- 1초 인트로 카드
- 항상 같은 제목 박스
- 같은 자막 등장감
- 깔끔한 교육용 일러스트 배경

## 제거된 요소

- 브랜드명 `LIM LANG`
- 로고 텍스트 `LL`
- 좌상단 로고 배지

## 핵심 특징

- OpenAI API 사용 안 함
- API Key 필요 없음
- 대본은 내장 표현 데이터 또는 직접 입력 사용
- 음성은 무료 `gTTS` 사용
- 배경은 **깔끔한 교육용 그림체**로 고정
- 주제별 오브젝트만 바뀌고 전체 스타일은 항상 동일

## Streamlit Cloud 배포

1. GitHub에 이 폴더 안의 파일을 전부 업로드합니다.
2. Streamlit Community Cloud에서 repo를 연결합니다.
3. Main file path를 `app.py`로 설정합니다.
4. Deploy를 누릅니다.

Secrets 설정은 필요 없습니다.

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 선택 파일

아래 파일은 없어도 됩니다.

```txt
assets/bgm.mp3
assets/bg_loop.mp4
```

`assets/bg_loop.mp4`가 있으면 그 영상이 배경으로 우선 사용됩니다.  
없으면 **깔끔한 교육용 고정 일러스트 배경**이 자동 생성됩니다.
